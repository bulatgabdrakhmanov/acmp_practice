Основная проблема задачи - необходимость в постоянной вставке/удалении элементов. Массивы с этим справляются
плохо, поэтому было принято решения использовать двусвязный список. Но у списка тоже есть минусы, чтобы
вставить/удалить элемент, листу необходимо добраться до нее через все остальные элементы. Эту проблему можно
решить если использовать вставку/удаление из конца или начала (листу не нужно до них добираться,
их адреса хрянятся в переменной). Для этого я создал 2 листа, один содержит длины строк до текущей включительно,
другой длины строк от текущей. Такое возможно, потому что структура задачи предполагает взаимодействие
только с текущей строкой и двумя ее соседями с двух сторон.
