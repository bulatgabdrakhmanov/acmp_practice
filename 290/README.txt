Для решения задачи необходимо представить план базы и учаток пустыни в матричном виде.
Затем необходимо сравнить эти два массива следующим способом:

План бызы:          Участок пустыни:
б1 б2               у1 у2 у3 у4
б3 б4               у5 у6 у7 у8

[ у1 == б1 && у2 == б2 && у5 == б3 && у6 == б4 ]
[ у2 == б1 && у3 == б2 && у6 == б3 && у7 == б4 ]
[ у3 == б1 && у4 == б2 && у7 == б3 && у8 == б4 ]

В ответ записывается кол-во наборов [], для которых верны все сравнения
